# Source and release policy

Every displayed rate, bracket, dollar amount, and budget share must cite a primary government source with publisher, title, URL, tax or fiscal year, access date, and exact table or page when available.

Derived values must record their formula and denominator. Source categories must remain visible even when mapped to a cross-state comparison category. Draft or provisional data must never be presented as reviewed.

Data releases are immutable. A correction creates a new patch release and a changelog entry. Each release must pass bracket-boundary tests, percentage-total checks, source-link checks, and a production build.
