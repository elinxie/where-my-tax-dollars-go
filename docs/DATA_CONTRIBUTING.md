# Data contribution guide

For each proposed state or locality, create a dated draft package and include:

- jurisdiction, tax year, fiscal year, and coverage status;
- official publisher, document title, URL, access date, and exact table/page;
- the raw value, denominator, unit, and any formula used;
- known gaps, assumptions, and source confidence;
- bracket-boundary and percentage-total checks.

Only primary government data may drive an on-screen tax amount or spending percentage. Secondary sources can help find a document but cannot be the final citation. Do not replace a published package in place: issue a new patch release with a changelog entry.
