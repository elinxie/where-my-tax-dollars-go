# Changelog

This project uses semantic versions. Data-source revisions are recorded separately from application releases.

## 0.1.0 — 2026-07-21

- Added a privacy-first 2025 tax estimator for W-2 households in Santa Clara County.
- Separated federal income, California income, employee payroll, and optional property taxes.
- Added federal, California, and county budget allocation views.
- Added plain-language explanations for school buses, street cleaning, roads, health care, schools, and safety-net programs.
- Added primary government source links, assumptions, exclusions, and educational-use disclaimer.
- Added responsive styling and social-sharing metadata.
