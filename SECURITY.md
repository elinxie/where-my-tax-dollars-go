# Security policy

Please do not post security vulnerabilities in public issues. Report them privately to the repository owner through GitHub's private vulnerability reporting feature, if enabled, or by opening a private contact route with the maintainer.

Do not include tax returns, addresses, Social Security numbers, employer records, API tokens, or other personal information in any report. The project is designed to process inputs locally in the browser and should not collect this data.

Supported version: the latest release on `main`.
